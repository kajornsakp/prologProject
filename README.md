##Super Pacman
###Description
This project is belong to Artificial Intelligent course by using Python and Prolog languange

###Features
- same as original pacman
- 4 abilities after each ghost die
	- laser gun
	- trap
	- sticky trap
	- blind

###Requirements
- Python (2.7.12)
- SWI-Prolog (7.3.32-devel)
- PySwip (0.2.3)
